2. In this lecture, we will...
  • Class Objectives
    • Understand modern web service structure and backend development
    • Practice hands-on backend frameworks (Node.js, Express, Spring Boot, FastAPI, Flask)
    • Experience with authentication and security (OAuth2, JWT, Firebase) and deployment automation (CI/CD, Docker, Kubernetes)
    • Proficiency with DB/ORM (MySQL, PostgreSQL, MongoDB, Redis, JPA, MyBatis, Prisma, SQLAlchemy)
  • Course Content (Key Learning Topics)
    • How web services are structured and designed
    • REST API design and authentication and documentation (Swagger, Postman)
    • Utilizing DB/RDB/NoSQL and ORMs
    • Communication protocols (WebSocket, MQTT)
    • Deployment and operations (CI/CD, Docker, Kubernetes)
    • Project-based labs (individual/team)

  • What you'll learn in each week
    • Weeks1-3: Web service structure, server types, and architecture design
    • Weeks 4-5: Node.js & Express, HTTP/REST APIs, Authentication (JWT), Swagger-Postman
    • Weeks 6-7: Learning backend frameworks (Spring Boot, Flask, FastAPI)
    • Week 8: Finalize Term Project I (build API server)
    • Weeks 9-10: DB/RDB/NoSQL + ORM, OAuth2 - Social Login (Google/Cakao), Firebase integration
    • Week 11: Communication protocols (WebSocket, MQTT) + Term Project II due
    • Weeks 12-14: Service maintenance, testing-logging-documentation, CI/CD, Docker, Kubernetes
    • Week 15: Term Project III due (server deployment)
  • Tips for taking the class
    • Focus on practice over theory
    • All assignments require GitHub commit/Swagger documentation
    • Projects can be utilized for personal portfolio
    • Active participation and other activities (competitions, seminars, open source activities) encouraged!
