Part 1 · Orientation

공공데이터포털 학습 의의

공공데이터포털 기반 학습의 핵심 의미는 실제 행정 데이터를 활용한 문제 정의, 데이터 해석, 서비스 구현의 통합 경험

현실 문제 기반 데이터 학습 환경 : 교통, 안전, 복지, 문화, 환경, 행정 분야의 실제 데이터 기반 프로젝트 설계 가능성
출처가 분명한 공공 데이터 자원 : 제공기관, 갱신주기, 관리부서, 이용 조건 확인이 가능한 신뢰성 높은 데이터 환경
파일데이터와 오픈API의 동시 경험 가능성 : 정적 데이터 분석과 동적 데이터 서비스 구현을 함께 다룰 수 있는 교육 구조
오픈소스 도구와의 높은 결합성 : Python, pandas, requests, folium, Streamlit, GitHub와 자연스럽게 연결되는 실습 구조
핵심 메시지: 공공데이터포털 학습의 목적은 데이터 다운로드 자체가 아니라 데이터를 설명 가능한 서비스로 전환하는 역량 형성

Part 3 · Reason

공공데이터포털 활용 이유

수업 소재 측면

학생이 체감 가능한 지역 문제, 생활 문제, 행정 문제와 직접 연결되는 주제 자원

기술 학습 측면

단순 문법 연습을 넘어 외부 데이터 소스 연결, 구조 해석, 시각화까지 이어지는 실전형 학습 구조

프로젝트 측면

학기 프로젝트, 팀 프로젝트, 캡스톤 아이디어로 확장 가능한 주제 다양성

이론과 실습의 연결성 : 데이터 구조, 네트워크 요청, JSON/XML, 예외 처리, 사용자 인터페이스를 하나의 사례로 묶을 수 있는 연결성
재현 가능한 실습 구조 : 동일 포털, 동일 문서, 동일 엔드포인트 기반의 반복 가능한 실습 환경
공공성 기반 스토리텔링 가능성 : 단순 시각화보다 서비스의 필요성과 사용자를 설명하기 쉬운 프로젝트 환경

Part 15 · Prompt Example

바이브 코딩 프롬프트 예시

전주시 가로등 정보 서비스를 활용한 Streamlit 앱을 Python으로 만들어줘.

기술 스택:
- Python
- Streamlit
- SQLite
- requests
- pandas

요구사항:
- 공공데이터포털의 전주시 가로등 정보 서비스 API를 사용
- 환경변수 이름은 JEONJU_API_KEY 사용
- API에서 받은 XML 데이터를 파싱해서 pandas DataFrame으로 변환
- SQLite 데이터베이스 파일 이름은 jeonju_lights.db 사용
- streetlamp 테이블 생성 후 데이터 저장
- Streamlit 화면에서 분전함명 검색, 건수 지표, 지도 시각화, 테이블 조회, CSV 다운로드 기능 제공
- API 오류 시 resultCode와 resultMsg를 사용자에게 보여주는 구조
- 코드 파일은 app.py 하나로 먼저 작성
- 초보 학생이 읽을 수 있게 함수 단위로 나누고 주석은 최소한으로 작성

출력 형식:
- 실행 가능한 전체 코드
- 필요한 pip 패키지 목록
- 실행 방법 설명

이 프롬프트의 장점

  • 기술 스택과 출력물이 명확한 구조
  • 저장소와 UI 요구사항이 동시에 포함된 구조
  • 학생 수준을 고려한 난이도 제어 구조

핵심 포인트
